# Various_graphs_and_charts_using_Python-R-Tableau
Using Tableau, Python and R, this project visualizes a series of different types of charts and graphs including: 
 
•	Tree Map
•	Area Chart 
•	Stacked Area Chart
•	Step Chart
•	Histogram
•	Box Plot
•	Bullet Chart
•	Spatial Map
•	Violin
•	Lollipop
•	Heat Map
•	Spatial Chart
•	Contour Chart
•	Bubble Chart
•	Scatter Plot
•	Density Map
